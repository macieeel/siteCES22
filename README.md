## Como rodar a aplicação:

No terminal, clone o projeto:

```
git clone https://github.com/macieeel/siteCES22.git
```

Entre na pasta criada:
```
cd siteCES22
```

Baixe os módulos:
```
pip install Flask
```

Por fim, execute o arquivo app.py:

```
python app.py
```
